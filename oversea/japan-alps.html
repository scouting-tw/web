日本北阿爾卑斯山脈極限挑戰

立山劍岳、槍岳穗高岳縱走完全攻略

行程總覽 重要提示 裝備與準備 每日行程 圖集錦囊

行程總覽

這是一趟為期12天，深入日本北阿爾卑斯心臟地帶的史詩級探險。行程將帶您挑戰極具代表性的立山連峰、險峻的劍岳，以及從槍岳到北穗高岳的經典縱走路線，包括著名的「大切戶」險境。這不僅是對體能的極限考驗，更是對高山經驗和技巧的全面檢閱，但回報將是無與倫比的壯麗景色與深刻的登山記憶。

  • 行程天數：12天
  • 核心區域：立山連峰、劍岳、上高地、槍岳、穗高岳 (含大切戶)
  • 難度級別：極高難度，具高風險性
  • 行程亮點：攀登「岩與雪的殿堂」劍岳，穿越日本阿爾卑斯最險峻的「大切戶」岩稜，登頂多座三千米級名峰，體驗立山黑部阿爾卑斯山脈路線的奇景，並沉浸在上高地的絕美溪谷風光中。

槍岳山頂標誌性的槍尖與遠方連綿山脈

行前重要提示 (請務必閱讀)

極高難度與風險： 此行程包含攀登劍岳和穿越大切戶 (Daikiretto)，這兩段都是日本登山路線中頂級的困難和高風險路段，需要豐富的高山徒步和攀岩經驗、良好的體能、優越的平衡感以及不怕暴露感 (懼高症者不宜)。務必在有經驗的嚮導陪同下或自身具備充足能力和準備的情況下進行。

  • 天氣至上： 高山天氣變化莫測。惡劣天氣 (強風、雨雪、濃霧) 會使這些高難度路段的風險急劇增加。務必每日確認天氣預報，若天氣不佳，應果斷放棄或更改行程。
  • 山屋預訂： 所有山屋 (山莊/小屋) 都必須提前預訂，尤其是在旺季 (7月中旬至9月)。部分熱門山屋可能需要提前數月預訂。大部分山屋僅接受現金。
  • 體能準備： 請務必進行充分的體能訓練，包括長距離徒步、海拔爬升和負重訓練。
  • 交通： 日本交通便利但複雜，提前查詢時刻表並規劃好接駁。考慮購買相關的交通周遊券。
  • 行李托運： 可以利用日本的行李托運服務 (如黑貓宅急便) 將不需要帶上山的行李從一個城市寄到下一個城市。
  • 保險： 務必購買包含高山救援的海外旅遊平安險。
  • 無痕山林 (LNT)： 遵守LNT原則，帶走所有垃圾。

裝備與準備

一份周全的裝備清單是安全完成此行的基礎。以下為建議裝備：

  • 專業登山鞋 (已磨合)、登山背包 (合適容量)。
  • 分層穿著衣物：排汗內層、保暖中層、防水防風硬殼外層。
  • 頭燈及備用電池 (部分路段需早出或摸黑)。
  • 安全頭盔： 攀登劍岳和穿越大切戶時強烈建議甚至必需。
  • 手套 (保護及保暖)、帽子 (保暖及遮陽)。
  • 登山杖、地圖、指南針/GPS。
  • 充足的水和高能量行動糧。
  • 急救包、防曬用品、太陽鏡。
  • 睡袋內襯 (山屋提供被褥)。

登山裝備平鋪展示圖，包含頭盔、登山杖等

每日詳細行程規劃

第一部曲：立山連峰與劍岳 (Tateyama & Mt. Tsurugi)

---

D1: 台北 ~ 東京 ~ 富山 ~ 室堂 (Murodo)

立山室堂的御庫裏池與立山倒影

路線/主要地點： 台北桃園機場 (TPE) → 東京成田/羽田機場 (NRT/HND) → 東京車站 → 富山站 → 立山站 → 美女平 (Bijodaira) → 室堂 (Murodo)。

交通/徒步簡述： 全天以國際與日本國內交通接駁為主，最終抵達海拔2450公尺的室堂平。

詳細說明：

  • 上午： 搭乘早班飛機從台北飛往東京。辦理入境後，前往東京車站搭乘北陸新幹線至富山 (約2-2.5小時)。
  • 下午： 從富山站轉乘富山地方鐵道至立山站 (約1小時)。 在立山站購買立山黑部阿爾卑斯山脈路線票券。接著搭乘立山登山纜車至美女平 (7分鐘)，再換乘立山高原巴士至室堂 (約50分鐘)，欣賞沿途高山景致。
  • 傍晚： 抵達室堂，辦理山屋入住。若時間許可，可至御庫裏池散步。

住宿： 室堂區域的山屋 (如：室堂山莊)。

餐食： 飛機餐/車站便當；山屋晚餐。

注意事項： 長途交通日，注意各段銜接。室堂海拔高，緩慢行動，多喝水以適應高山環境。

D2: 室堂 (Murodo) ～ 劍山莊 (Kenzanso) ～「劍岳 (Mt. Tsurugi)」～ 劍山莊

攀登劍岳時的鎖鏈或鐵梯險峻路段

路線/主要地點： 室堂 → 雷鳥澤營地 → 別山乘越 → 劍御前小舍 → 劍山莊 → 劍岳山頂 → 劍山莊。

交通/徒步簡述： 上午徒步至劍山莊，下午輕裝挑戰日本百名山中最險峻之一的劍岳。

詳細說明：

  • 上午： 室堂出發，經雷鳥澤、別山乘越、劍御前小舍，徒步至劍山莊 (約3-4小時，海拔爬升約300-350公尺)。 沿途欣賞立山連峰與劍岳雄姿。抵達後辦理入住並輕裝。
  • 下午： 從劍山莊往返登頂劍岳 (海拔2999公尺)。此路段極具挑戰，包含多段鎖鏈、鐵梯及狹窄岩脊 (約4-6小時往返，從劍山莊爬升約210公尺，但難度在於地形)。
警告： 攀登劍岳務必佩戴頭盔，並在天氣良好、自身狀態極佳時進行。量力而為，安全第一。

住宿： 劍山莊 (Kenzanso)。

餐食： 山屋早晚餐，行動糧/山屋午餐便當。

注意事項： 出發前務必向山屋確認劍岳最新路況。若不登頂，可在劍山莊周邊賞景。

D3: 劍山莊 (Kenzanso) ～「立山主峰稜線 (Tateyama Main Ridge)」～ 雷鳥莊 (Raichoso)

立山主峰稜線上行走，背景是開闊的高山景色

路線/主要地點： 劍山莊 → 劍御前小舍 → 別山 → 真砂岳 → 富士折立 → 大汝山 → 雄山 → 一之越山莊 → 雷鳥莊。

交通/徒步簡述： 全天進行立山主峰稜線縱走，登頂立山三山，最後下榻溫泉山屋雷鳥莊。

詳細說明：

  • 上午： 從劍山莊返回劍御前小舍，開始沿立山主峰稜線向南縱走，途經別山(可選)、真砂岳、富士折立(2999m)，登頂立山最高峰大汝山(3015m)，再至雄山(3003m，山頂有神社)。此段約5-7小時。
  • 下午： 從雄山山頂下撤至一之越山莊，再續下坡返回室堂平，前往雷鳥莊 (約1.5-2小時)。

住宿： 雷鳥莊 (Raichoso，以溫泉聞名)。

餐食： 山屋早晚餐，行動糧/山屋午餐便當。

注意事項： 全程高海拔稜線，注意防風防曬。雄山神社參拜可能需付費。

D4: 雷鳥莊 (Raichoso) ～ 室堂 ～ 黑部水庫 (Kurobe Dam) ～ 扇澤 (Ogizawa) ～ 松本 (Matsumoto)

黑部水庫洩洪的壯觀場面或立山空中纜車

路線/主要地點： 雷鳥莊 → 室堂 → 大觀峰 → 黑部平 → 黑部湖/黑部水庫 → 扇澤 → 信濃大町站 → 松本。

交通/徒步簡述： 體驗立山黑部阿爾卑斯山脈路線的各種交通工具，橫斷山脈後前往松本市。

詳細說明：

  • 上午： 從雷鳥莊步行至室堂總站 (約20-30分鐘)。 接續搭乘立山隧道無軌電車至大觀峰 (10分鐘)，轉立山空中纜車至黑部平 (7分鐘)，再換黑部登山纜車至黑部湖 (5分鐘)。步行參觀日本最大的拱形水壩——黑部水庫。
  • 下午： 從黑部水庫搭乘關電隧道無軌電車至扇澤 (16分鐘)。 轉乘路線巴士至JR信濃大町站 (約40分鐘)，再搭JR大糸線電車至松本 (約1小時)。
  • 傍晚： 抵達松本，辦理酒店入住。可逛市區或觀賞松本城夜景。

住宿： 松本市的酒店。

餐食： 山屋早餐；沿途簡餐；松本晚餐。

注意事項： 複雜交通日，務必預先查詢各段時刻表。

第二部曲：上高地與槍岳、穗高岳縱走 (Kamikochi, Mt. Yari & Mt. Hotaka)

---

D5: 松本 (Matsumoto) - 上高地 (Kamikochi) - 橫尾山莊 (Yokoo Sanso)

上高地河童橋與穗高連峰的經典畫面

路線/主要地點： 松本 → 新島島站 → 上高地巴士總站 → 河童橋 → 明神 → 德澤 → 橫尾山莊。

交通/徒步簡述： 從松本前往有「神之故鄉」美譽的上高地，開始沿梓川向上游徒步至橫尾山莊。

詳細說明：

  • 上午： 從松本站搭乘松本電鐵上高地線至新島島站 (約30分鐘)，轉路線巴士至上高地巴士總站 (約1小時)。
  • 下午： 抵達上高地 (約1500m) 後開始徒步，經河童橋、明神、德澤，最終抵達橫尾山莊 (約1620m)。此段路程約3-3.5小時，平緩好走，景色優美。

住宿： 橫尾山莊 (Yokoo Sanso)。

餐食： 松本早餐；上高地或沿途簡餐；橫尾山莊晚餐。

注意事項： 上高地有交通管制。橫尾山莊是重要中繼點，床位需預訂。

D6: 橫尾山莊 (Yokoo Sanso) - 槍沢 (Yarisawa) - 槍岳山莊 (Yarigatake Sanso)

槍岳的尖銳峰頂特寫

路線/主要地點： 橫尾山莊 → 一ノ俣 → 槍沢ロッヂ → 槍沢露營場 → 槍岳山莊。

交通/徒步簡述： 全天為具有挑戰性的爬升，沿槍沢溪谷上溯至槍岳主峰下的槍岳山莊。

詳細說明：

  • 全天： 從橫尾山莊出發，經槍沢ロッヂ、槍沢露營場，一路爬升至槍岳山莊 (約3060m)。全程約5-7小時，海拔爬升約1440公尺。 後半段坡度陡峭。
  • 下午/傍晚可選： 若體力、天氣和時間許可，可輕裝登頂槍岳 (3180m)，山頂路段有鎖鏈鐵梯，需小心 (往返約1-1.5小時)。

住宿： 槍岳山莊 (Yarigatake Sanso)。

餐食： 山屋早晚餐，行動糧/山屋午餐便當。

注意事項： 持續爬升辛苦。槍沢上部初夏可能尚有殘雪。

D7: 槍岳山莊 (Yarigatake Sanso) - 南岳小屋 (Minamidake Koya) - 北穗高小屋 (Kitahotaka Koya)

大切戶驚險的岩稜或鎖鏈路段

路線/主要地點： 槍岳山莊 → 大喰岳 → 中岳 → 南岳 → 南岳小屋 → 大切戶 (Daikiretto) → 長谷川ピーク → 北穗高岳 → 北穗高小屋。

交通/徒步簡述： 全日本阿爾卑斯山脈中最具挑戰性的岩稜縱走之一，穿越著名的「大切戶」。

詳細說明：

  • 上午： 若前日未登頂槍岳，可清晨攻頂。早餐後，從槍岳山莊出發，經大喰岳、中岳，抵達南岳 (3033m) 及南岳小屋 (約3-4小時)。
  • 下午： 從南岳小屋開始挑戰**大切戶**，經長谷川ピーク、A沢のコル、飛騨泣き，登頂北穗高岳 (3106m)，抵達北穗高小屋 (約3100m)。此段約4-6小時。
極高風險警告： 大切戶是極度險峻的斷崖岩稜，全程充滿鎖鏈、鐵梯、垂直岩壁，暴露感極強。僅適合經驗豐富、無懼高症、體能技術俱佳者，並務必在良好天氣下進行。強烈建議佩戴頭盔。

住宿： 北穗高小屋 (Kitahotaka Koya)。

餐食： 山屋早晚餐，行動糧/山屋午餐便當。

注意事項： 若無把握或天氣不佳，絕對不要嘗試大切戶。南岳小屋是最後撤退點。

D8: 北穗高小屋 (Kitahotaka Koya) - 涸沢 (Karasawa) - 德澤 (Tokusawa)

涸沢冰斗圈谷的壯麗景色，秋季可為紅葉

路線/主要地點： 北穗高小屋 → 涸沢坳 → 涸沢小屋/涸沢ヒュッテ → 本谷橋 → 橫尾山莊(經過) → 德澤園/德澤ロッヂ。

交通/徒步簡述： 從險峻的北穗高岳下撤至優美的涸沢冰斗，再續下山至德澤的森林溪谷。

詳細說明：

  • 上午： 從北穗高小屋出發，陡峭下行至涸沢坳，再到涸沢小屋/涸沢ヒュッテ (約2.5-3.5小時)。 此段路多碎石，需小心。
  • 下午： 從涸沢繼續下山，經本谷橋、橫尾山莊，抵達德澤 (約3-4小時)。

住宿： 德澤園 (Tokusawa-en) 或 德澤ロッヂ (Tokusawa Lodge)。

餐食： 山屋早晚餐，涸沢小屋或行動糧午餐。

注意事項： 從北穗高下涸沢的路段需非常小心。

D9: 德澤 (Tokusawa) - 河童橋 - 大正池 (Taisho Pond) - 田代池 (Tashiro Pond) ~ 上高地 (Kamikochi)

大正池中枯木與穗高連峰的夢幻景色

路線/主要地點： 德澤 → 明神 → 河童橋 → 田代池 → 大正池 → 上高地巴士總站。

交通/徒步簡述： 在上高地進行經典的平緩徒步，遊覽各主要景點。

詳細說明：

  • 上午： 從德澤輕鬆徒步至明神 (可選遊明神池)，再到河童橋 (約1.5-2小時)。
  • 中午： 在河童橋附近午餐並遊覽。
  • 下午： 從河童橋沿梓川徒步前往田代池、大正池，欣賞夢幻景色。之後返回上高地巴士總站 (河童橋至大正池往返約2-3小時)。

住宿： 上高地內酒店/旅館，或當日離開。行程註明「上高地解散」。

餐食： 山屋早餐；上高地午餐；晚餐自理。

第三部曲：白馬與返程

---

D10: 上高地解散 ~ 白馬村 (Hakuba Village)

夏季白馬村遠眺白馬三山的青翠景色

路線/主要地點： 上高地巴士總站 → 平湯溫泉 → 松本 → 白馬村。

交通/徒步簡述： 從上高地出發，經多次交通轉換，前往滑雪勝地白馬村。

詳細說明：

  • 上午： 從上高地巴士總站搭巴士至平湯溫泉 (約25分鐘)。
  • 中午/下午： 從平湯溫泉轉巴士至松本 (約1.5小時)，再從松本站搭JR大糸線電車至白馬站 (約1小時)。 抵達白馬村後辦理入住，可初步探索村莊。

住宿： 白馬村的酒店或旅館。

餐食： 上高地或平湯溫泉早午餐；白馬村晚餐。

注意事項： 交通轉換日，預先查好時刻表。

D11: 白馬村 (Hakuba Village) - 松本 (Matsumoto)

松本城的天守閣與護城河

路線/主要地點： 白馬村 → 松本市 (松本城、繩手通、中町通)。

交通/徒步簡述： 上午在白馬村周邊活動，下午返回松本市觀光。

詳細說明：

  • 上午： 白馬村周邊活動，如搭乘纜車上山短途徒步 (例如前往八方池，視季節和纜車開放情況)，或參觀白馬跳台滑雪競技場。
  • 中午/下午： 從白馬站搭JR大糸線電車返回松本 (約1小時)。 抵達後辦理入住，下午遊覽松本城、繩手通、中町通。

住宿： 松本市的酒店。

餐食： 白馬村早午餐；松本晚餐。

D12: 東京 ~ 台北

東京晴空塔或繁忙的澀谷十字路口

路線/主要地點： 松本 → 東京 (新宿/東京車站) → 東京成田/羽田機場 → 台北桃園機場。

交通/徒步簡述： 全天以交通為主，從松本返回東京，再搭機返台。

詳細說明：

  • 上午： 從松本站搭JR中央本線特急列車「梓(Azusa)」或「超級梓(Super Azusa)」直達東京新宿站或東京站 (約2.5-3小時)。
  • 中午/下午： 從東京市區搭乘成田特快或利木津巴士前往機場，辦理登機手續，搭機返回台北。

餐食： 松本早餐；東京或機場午餐；飛機餐。

注意事項： 預留充足的交通及機場手續時間。
